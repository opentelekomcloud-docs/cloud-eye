Alarm Rules
===========

The **Alarm Rules** page displays all alarm rules in a resource group. You can enable, disable, modify, or delete an alarm rule.

Procedure
---------

#. Log in to the management console.

2. In the upper left corner, select a region and project.
3. Choose **Service List** > **Cloud Eye**.
4. In the navigation pane on the left, choose **Resource Groups**.
5. Click a resource group name to go to the **Resource Overview** page.
6. In the navigation pane, choose **Alarm Rules** to view all alarm rules in the resource group.

On the **Alarm Rules** page, you can quickly create alarm rules for resources in the resource group. For details, see :ref:`Creating an Alarm Rule <en-us_topic_0084572213>`.

.. note::

   For event alarm rules, you can set the monitoring scope to **Resource groups** only when the event source is DDS, RDS, or DCS.
