Viewing Event Monitoring Data
=============================

Scenarios
---------

This topic describes how to view the event monitoring data.

Procedure
---------

#. Log in to the management console.

#. Choose **Service List** > **Cloud Eye**.

#. In the navigation pane on the left, choose **Event Monitoring**.

   On the displayed **Event Monitoring** page, all system events generated in the last 24 hours are displayed by default.

   You can view events in the last 1 hour, last 3 hours, last 12 hours, last 24 hours, last 7 days, or last 30 days. Alternatively, you can set a custom time range to view events triggered within that period.


   .. figure:: /_static/images/en-us_image_0000001693521761.png
      :alt: **Figure 1** Event Monitoring

      **Figure 1** Event Monitoring

#. In the row containing the target event, click **View Graph** in the **Operation** column. Then, you can view the monitoring data of last 24 hours.

   You can view monitoring data of a specified event in the last 1 hour, last 3 hours, last 12 hours, last 1 day, last 7 days, or last 30 days. Alternatively, you can set a custom time range by specifying the start time and end time to view monitoring data of a specified event within that period.


   .. figure:: /_static/images/en-us_image_0000002464270489.png
      :alt: **Figure 2** Viewing graphs

      **Figure 2** Viewing graphs

#. Click **View Event** in the **Operation** column to view details about a specific event.

#. Above the event list, select a property type and enter a key word to filter the desired event.

#. To view events of a specific time period, click the corresponding bar chart.

#. In the event list, select multiple events and click the arrow in the lower right corner to view their monitoring details.


   .. figure:: /_static/images/en-us_image_0000002464136293.png
      :alt: **Figure 3** Viewing event monitoring details in batches

      **Figure 3** Viewing event monitoring details in batches
