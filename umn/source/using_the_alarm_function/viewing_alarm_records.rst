Viewing Alarm Records
=====================

The **Alarm Records** page displays the status changes for all alarm rules, making it easy to track and review them in one place. By default, alarm records of the last seven days are displayed. You can select a time range to display alarm records in the past 180 days.

When an alarm is generated, you can view the alarm records about the cloud resource.


Viewing Alarm Records
---------------------

#. Log in to the management console.

#. In the upper left corner, select a region and project.

#. Choose **Service List** > **Cloud Eye**.

#. Choose **Alarm Management** > **Alarm Records**.

   On the **Alarm Records** page, you can view the status changes of all alarm rules in the last 7 days.

#. Click **View Details** in the **Operation** column. On the displayed drawer, view the basic information about the resource, and view the data that triggered the latest alarm status change.

   .. note::

      On the Alarm Records page, click **Export** in the upper left corner to export alarm records.

Filtering Alarm Records
-----------------------

-  You can select a time range within the past 180 days to view alarm records.
-  On the **Alarm Records** page, you can select a property type and enter a value to filter alarm records by status, alarm severity, alarm rule name, resource type, resource ID, or alarm rule ID.
