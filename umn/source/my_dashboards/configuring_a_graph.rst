Configuring a Graph
===================

If the existing metrics do not meet your requirements, you can add, modify, and delete metrics on a line chart and a bar chart.

Procedure for Configuring Line Charts
-------------------------------------

#. Log in to the management console.
#. In the upper left corner, select a region and project.
#. Choose **Service List** > **Cloud Eye**.

4. In the navigation pane, choose **My Dashboards**.

5. Locate the dashboard for which you want to configure a graph and click its name.

6. Click |image1| to copy\ :ref:`, delete, <ces_01_0232>` or edit, the graph, or change its legend name.

   a. To copy a graph, locate the graph and click **Copy**. In the displayed dialog box, set parameters.

      -  Copying a graph to the current dashboard: Set **From** to **Current Dashboard**, set **Graph Name**, and click **OK**.
      -  Copying a graph to another dashboard: Set **From** to **Other dashboards**. Set **Enterprise Project**, **Target Dashboard**, and **Graph Name**. Then, click **OK**.

   b. To edit a graph, locate the graph and click **Edit**. On the displayed page, modify **Monitoring Scope** and **Advanced Settings**.
   c. To change the legend name, click **Change Legend Name**. In the displayed dialog box, enter a legend name in the **Legend** column and click **OK**.

      :alt: **Figure 1** Managing a graph

      **Figure 1** Managing a graph

Procedure for Configuring Bar Charts
------------------------------------

#. Log in to the management console.
#. In the upper left corner, select a region and project.
#. Choose **Service List** > **Cloud Eye**.

4. In the navigation pane, choose **My Dashboards**.
5. Locate the dashboard for which you want to configure a graph and click its name.

6. Click |image2| to configure **Quantity** and **Sorting Order**.


   .. figure:: /_static/images/en-us_image_0000001835931196.png
      :alt: **Figure 2** Sorting metrics

      **Figure 2** Sorting metrics

7. Click |image3| to copy, edit, or :ref:`delete <ces_01_0232>` the graph.

   a. To copy a graph, locate the graph and click **Copy**. In the displayed dialog box, set parameters.

      -  Copying a graph to the current dashboard: Set **From** to **Current Dashboard**, set **Graph Name**, and click **OK**.
      -  Copying a graph to another dashboard: Set **From** to **Other dashboards**. Set **Enterprise Project**, **Target Dashboard**, and **Graph Name**. Then, click **OK**.

   b. To edit a graph, locate the graph and click **Edit**. On the displayed page, modify **Monitoring Scope**, **Quantity**, and **Advanced Settings**.
