Modifying a Resource Group
==========================

Function
--------

This API is used to modify a resource group.

URI
---

PUT /v2/{project_id}/resource-groups/{group_id}

.. table:: **Table 1** Path parameters

   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------+
   | Parameter       | Mandatory       | Type            | Description                                                                                                                    |
   +=================+=================+=================+================================================================================================================================+
   | project_id      | Yes             | String          | Project ID.                                                                                                                    |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | For details about how to obtain the project ID, see :ref:`Obtaining a Project ID <ces_03_0057>`.                               |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Minimum: **1**                                                                                                                 |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Maximum: **64**                                                                                                                |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | **Regular expression matching**: ^[a-zA-Z0-9-]{1,64}$                                                                          |
   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------+
   | group_id        | Yes             | String          | Resource group ID. The value starts with **rg** and is followed by 22 characters of letters, digits, or a combination of both. |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Minimum: **24**                                                                                                                |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Maximum: **24**                                                                                                                |
   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------+
