Batch Deleting Associated Resources from a Custom Resource Group
================================================================

Function
--------

This API is used to batch delete associated resources from a custom resource group whose **type** is **Manual**.

URI
---

POST /v2/{project_id}/resource-groups/{group_id}/resources/batch-delete

.. table:: **Table 1** Path parameters

   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------+
   | Parameter       | Mandatory       | Type            | Description                                                                                                                    |
   +=================+=================+=================+================================================================================================================================+
   | project_id      | Yes             | String          | Project ID.                                                                                                                    |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | For details about how to obtain the project ID, see :ref:`Obtaining a Project ID <ces_03_0057>`.                               |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Minimum: **1**                                                                                                                 |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Maximum: **64**                                                                                                                |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | **Regular expression matching**: ^[a-zA-Z0-9-]{1,64}$                                                                          |
   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------+
   | group_id        | Yes             | String          | Resource group ID. The value starts with **rg** and is followed by 22 characters of letters, digits, or a combination of both. |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Minimum: **24**                                                                                                                |
   |                 |                 |                 |                                                                                                                                |
   |                 |                 |                 | Maximum: **24**                                                                                                                |
   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------+

Example Requests
----------------

Batch deleting associated resources from a custom resource group

.. code-block::

   {
     "resources" : [ {
       "namespace" : "SYS.ECS",
       "dimensions" : [ {
         "name" : "instance_id",
         "value" : "4270ff17-aba3-4138-89fa-820594c39755"
       } ]
     } ]
   }

Example Responses
-----------------

**Status code: 200**

Resources deleted.

.. code-block::

   {
     "succeed_count" : 1
   }
