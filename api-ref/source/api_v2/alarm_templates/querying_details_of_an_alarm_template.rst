Querying Details of an Alarm Template
=====================================

Function
--------

This API is used to query details of an alarm template.

URI
---

GET /v2/{project_id}/alarm-templates/{template_id}

.. table:: **Table 1** Path parameters

   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Parameter       | Mandatory       | Type            | Description                                                                                                                                                        |
   +=================+=================+=================+====================================================================================================================================================================+
   | project_id      | Yes             | String          | Project ID.                                                                                                                                                        |
   |                 |                 |                 |                                                                                                                                                                    |
   |                 |                 |                 | For details about how to obtain the project ID, see :ref:`Obtaining a Project ID <ces_03_0057>`.                                                                   |
   |                 |                 |                 |                                                                                                                                                                    |
   |                 |                 |                 | Minimum: **1**                                                                                                                                                     |
   |                 |                 |                 |                                                                                                                                                                    |
   |                 |                 |                 | Maximum: **64**                                                                                                                                                    |
   |                 |                 |                 |                                                                                                                                                                    |
   |                 |                 |                 | **Regular expression matching**: ^[a-zA-Z0-9-]{1,64}$                                                                                                              |
   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | template_id     | Yes             | String          | ID of an alarm template. The ID starts with **at** and is followed by letters, digits, or a combination of both. The value can contain a maximum of 64 characters. |
