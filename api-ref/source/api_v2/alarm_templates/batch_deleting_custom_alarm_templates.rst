Batch Deleting Custom Alarm Templates
=====================================

Function
--------

This API is used to delete custom alarm templates in batches.

URI
---

POST /v2/{project_id}/alarm-templates/batch-delete

.. table:: **Table 1** Path parameters

   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------+
   | Parameter       | Mandatory       | Type            | Description                                                                                      |
   +=================+=================+=================+==================================================================================================+
   | project_id      | Yes             | String          | Project ID.                                                                                      |
   |                 |                 |                 |                                                                                                  |
   |                 |                 |                 | For details about how to obtain the project ID, see :ref:`Obtaining a Project ID <ces_03_0057>`. |
   |                 |                 |                 |                                                                                                  |
   |                 |                 |                 | Minimum: **1**                                                                                   |
   |                 |                 |                 |                                                                                                  |
   |                 |                 |                 | Maximum: **64**                                                                                  |
   |                 |                 |                 |                                                                                                  |
   |                 |                 |                 | **Regular expression matching**: ^[a-zA-Z0-9-]{1,64}$                                            |
   +-----------------+-----------------+-----------------+--------------------------------------------------------------------------------------------------+

Request Parameters
------------------

.. table:: **Table 2** Request header parameters

   +-----------------+-----------------+-----------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Parameter       | Mandatory       | Type            | Description                                                                                                                                                                                                                            |
   +=================+=================+=================+========================================================================================================================================================================================================================================+
   | X-Auth-Token    | Yes             | String          | User token. It is a response to the API for obtaining a user token. This API is the only one that does not require authentication. After a response is returned, the value of **X-Subject-Token** in the response header is the token. |
   |                 |                 |                 |                                                                                                                                                                                                                                        |
   |                 |                 |                 | Minimum: **1**                                                                                                                                                                                                                         |
   |                 |                 |                 |                                                                                                                                                                                                                                        |
   |                 |                 |                 | Maximum: **16384**                                                                                                                                                                                                                     |
   +-----------------+-----------------+-----------------+----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

.. table:: **Table 3** Request body parameters

   +------------------------+-----------------+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | Parameter              | Mandatory       | Type             | Description                                                                                                                                                                                                                                                                                                      |
   +========================+=================+==================+==================================================================================================================================================================================================================================================================================================================+
   | template_ids           | Yes             | Array of strings | IDs of the alarm templates to be deleted in batches. Alarm templates that are not associated with alarm rules can be deleted in batches. For alarm templates that are associated with alarm rules, you can delete only one alarm template at a time. If you delete multiple ones, an exception will be returned. |
   |                        |                 |                  |                                                                                                                                                                                                                                                                                                                  |
   |                        |                 |                  | Array length: **1** to **100**                                                                                                                                                                                                                                                                                   |
   +------------------------+-----------------+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+
   | delete_associate_alarm | Yes             | Boolean          | Whether alarm rules associated with an alarm template will be deleted when you delete the alarm template. **true** indicates that the alarm rules will be deleted. **false** indicates that only the alarm template will be deleted.                                                                             |
   +------------------------+-----------------+------------------+------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------+

Example Requests
----------------

Deleting custom alarm templates in batches

.. code-block::

     "template_ids" : [ "at1628592157541dB1klWgY6" ],
     "delete_associate_alarm" : false
   }
